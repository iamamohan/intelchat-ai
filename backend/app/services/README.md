# IntelChat

IntelChat is an AI-powered RAG (Retrieval-Augmented Generation) knowledge assistant.

## Features

- PDF Upload
- AI Chat
- Conversation Memory
- Ollama Integration
- ChromaDB Vector Search
- FastAPI Backend
- Next.js Frontend
- Streaming Responses
- Drag & Drop Upload
- Dark & Light Theme

## Tech Stack

Frontend
- Next.js
- React
- Tailwind CSS
- Framer Motion

Backend
- FastAPI
- Ollama
- ChromaDB
- Sentence Transformers
- SQLite

LLM
- Qwen2.5

Embedding Model
- all-MiniLM-L6-v2
